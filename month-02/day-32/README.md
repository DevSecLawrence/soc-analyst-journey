# Day 32 — EDR Fundamentals: Understanding Endpoint Detection

EDR came up in 7/10 job descriptions I looked at and I had zero knowledge of it going in. Spent today fixing that — researched CrowdStrike, Microsoft Defender for Endpoint, and Carbon Black, then mapped out exactly what EDR catches that SIEM misses and vice versa.

Main thing I learned: EDR doesn't replace SIEM. They see completely different things. You need both.
